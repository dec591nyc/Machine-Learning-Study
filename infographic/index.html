01 監督式學習

GBDT 家族 XGBoost / LightGBM

表格與結構化數據精準預測之王
採用漸進式提升決策樹架構，能處理高維表格特徵，抗噪能力極強。在結構化數據競賽中維持統治級的準確率。
台灣產業實務場景：
  • 🏭 半導體: 日月光封測製程良率預測與參數調校
  • 💳 金融科技: 富邦/國泰信用卡違約風險預測與核貸
  • 🏠 房地產: 信義房屋實價登錄自動估值系統 (AVM)
  • 🛒 零售電商: momo 購物網節慶銷量與庫存預測
工具庫: XGBoost, LightGBM 結構化首選
02 自監督 / 深度學習

Transformer BERT / GPT / Llama

多模態理解、注意力機制與文本生成
基於自注意力機制 (Self-Attention)，可建立極長文本的關聯，為現代大型語言模型與生成式 AI (GenAI) 的核心基石。
台灣產業實務場景：
  • 🏥 智慧醫療: 長庚醫院電子病歷自動摘要與醫病對答
  • 💳 金融科技: 智慧客服語意理解、法規合約自動審查
  • 🏭 半導體: 專利庫與內部技術知識庫 (RAG) 智慧檢索
工具庫: PyTorch, HuggingFace GenAI核心
03 監督式學習

邏輯回歸 Logistic Regression

高解釋性二分類之白盒標準模型
透過 Sigmoid 函數輸出機率，數學邏輯透明，各特徵係數直接對應勝算比 (Odds Ratio)，受法規監管部門高度青睞。
台灣產業實務場景：
  • 💳 金融科技: 聯徵中心信用評分卡 (Scorecard) 合規計算
  • 🏥 智慧醫療: 臨床醫學疾病致病風險顯著性分析
  • 🏠 房地產: 不動產買方購屋交易機率與意願預測
工具庫: Scikit-Learn, R 極高合規解釋
04 監督式學習

隨機森林 Random Forest

強健的分袋整合分類與特徵篩選器
透過多棵獨立決策樹投票，能有效降低變異數並避免過擬合，自帶特徵重要性 (Feature Importance) 計算功能。
台灣產業實務場景：
  • 🏭 半導體: 晶圓封裝機台關鍵特徵篩選、製程異常分類
  • 🏠 房地產: 地段價值影響因子(嫌惡設施/捷運)重要性排行
  • 🏥 智慧醫療: 多體學基因序列關鍵特徵因子篩選
工具庫: Scikit-Learn, Python 強健的Baseline
05 自監督 / 深度學習

卷積神經網路 CNN / YOLO / U-Net

電腦視覺、特徵提取與即時物體偵測
透過卷積層提取空間特徵，極度適用於圖像。在瑕疵定位、影像分割和即時目標檢測上表現最為優異。
台灣產業實務場景：
  • 🏭 半導體: 產線 **AOI 自動光學檢測**（晶圓瑕疵、PCB空焊）
  • 🏥 智慧醫療: 影像診斷（X光、CT腫瘤病灶分割與識別）
  • 🏠 房地產: 房仲網房屋照片（室內格局與裝潢）自動分類
工具庫: PyTorch, OpenCV 電腦視覺專精
06 非監督式學習

K-Means 分群 K-Means Clustering

無標籤數據分群與消費者畫像建模
計算樣本與質心間的歐幾里得距離，進行迭代聚類，能為雜亂無章的大量特徵快速打上分類標籤，是會員行銷核心。
台灣產業實務場景：
  • 🛒 零售電商: 蝦皮/momo 會員 **RFM 消費價值分群**與精準行銷
  • 💳 金融科技: 財富管理客戶風險偏好與資產配置族群劃分
  • 🏠 房地產: 全台都市生活圈地段與自動商圈群聚分析
工具庫: Scikit-Learn 客群畫像首選
07 自監督 / 深度學習

LSTM / GRU 長短期記憶網路

時間序列特徵學習與歷史依賴建模
透過遺忘門與輸入門機制解決傳統 RNN 梯度消失問題，特別適用於預測具有周期性與歷史關聯的連續時序數據。
台灣產業實務場景：
  • 🏭 半導體: **機台預防性維護 (PdM)** 振動與溫度異常趨勢預測
  • 💳 金融科技: 外匯匯率波動、股票市場價格時間序列預測
  • 🛒 零售電商: 智慧供應鏈每日倉儲物流銷量預測與庫存調配
工具庫: PyTorch, TensorFlow 時序預測精準
08 非監督式學習

孤立森林 Isolation Forest

基於分割空間路徑的非監督異常檢測
利用樹狀結構隨機切分數據。由於異常值極少且與正常值差異大，其在樹中會極快落入葉子節點（路徑極短），極佳的異常檢測效率。
台灣產業實務場景：
  • 🏭 半導體: 晶圓廠 **FDC 製程異常檢測**與儀器數值突變警示
  • 💳 金融科技: **洗錢防制 (AML)** 行為追蹤、信用卡即時盜刷偵測
  • 🛒 零售電商: 惡意搶票黃牛與帳號惡意洗單行徑偵測
工具庫: Scikit-Learn 非監督安全防護
09 強化學習

深度強化學習 PPO / DPO / DQN

環境反饋與多步決策獎勵最大化
Agent 與動態環境交互，透過反饋機制學習最大化累積回報的決策路徑。常用於控制、對齊（LLM 偏好對齊）與動態競價。
台灣產業實務場景：
  • 🏭 半導體: 晶圓廠自動化倉儲 **AGV 無人搬運車路徑避碰調度**
  • 🛒 零售電商: 即時廣告競價系統 (RTB) 最優定價策略優化
  • 💳 金融科技: 避險基金高頻演算法量化交易策略優化
工具庫: Ray/RLlib, PyTorch 多步決策
10 非監督式學習

主成分分析 PCA

特徵空間正交分解與大數據降維去噪
透過變異數最大化方向投影，將數百維的高維特徵壓縮至關鍵主成分，保留主要資料分布結構，去除高頻噪聲。
台灣產業實務場景：
  • 🏭 半導體: 多感測器參數特徵降維、**統計製程管制 (SPC)**
  • 🏥 智慧醫療: 基因體學與蛋白質體學特徵降維與視覺化
  • 💳 金融科技: 多因子量化選股模型中資產風險因子特徵提取
工具庫: Scikit-Learn, PCA 維度壓縮去噪
