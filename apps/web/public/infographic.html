📈 預測連續數值
監督式 #01

線性回歸
Linear Regression

擬合趨勢線以預測變數之間的連續數值關係。

台灣產業實務案例
🏠 信義房屋：中古屋估價系統 AVM
💡 使用情境

快速建立連續數值預測 Baseline，且需要清晰可解釋的特徵權重係數。

🎯 分類與預測標籤
監督式 #02

邏輯回歸
Logistic Regression

使用 S 型曲線預測二分類結果的機率值。

台灣產業實務案例：
💳 國泰世華信用卡盜刷預測
使用情境：法規高度合規監管，要求可解釋的勝算比 (Odds Ratio)。
監督式 #03

決策樹
Decision Tree

依據特定規則進行分裂，形成樹狀決策路徑。

台灣產業實務案例：
🏭 台積電晶片良率分類檢測規則
使用情境：決策邏輯必須轉換成直觀的 IF-THEN 人類可讀規則。
監督式 #04

隨機森林
Random Forest

集結多棵獨立決策樹投票，以降低過擬合。

台灣產業實務案例：
🏥 奇美醫院急診轉 ICU 預警
使用情境：表格數據有缺失值，需要穩定且強健的高精度 Baseline。
監督式 #05

支持向量機
SVM

在高維空間中，尋找最大間距分類界線。

台灣產業實務案例：
📶 聯發科晶片異常信號判定
使用情境：特徵維度高（如文字、基因）、樣本量中等且界線清晰。
監督式 #06

K-近鄰演算法
KNN

依據特徵空間中最鄰近的 K 個樣本投票。

台灣產業實務案例：
🛒 momo購物相似用戶購買推薦
使用情境：特徵維度低且資料乾淨，希望基於鄰近歷史相似點預測。
監督式 #07

單純貝氏
Naive Bayes

假設特徵皆獨立，基於貝氏機率公式分類。

台灣產業實務案例：
🛒 PChome客服信件分類分流
使用情境：高維度單字特徵（如語意/文本分類），需要極快運算。
🫧 自動資料分群
非監督式 #08

K-Means 分群
K-Means Clustering

迭代優化樣本與 K 個中心點的距離進行分類。

台灣產業實務案例
🏪 全家便利商店：會員RFM消費畫像分群
💡 使用情境

資料沒有標籤，需要自動發掘客群結構、定位特徵或進行商業劃分時。

🗜️ 降維 / 🧠 深度
非監督 #09

主成分分析 (PCA)
PCA

將高維數據正交投影至變異數最大方向。

台灣產業實務案例：
🏭 日月光封裝SPC監控
使用情境：維度過多需要特徵壓縮、去噪或二/三維視覺化。
深度學習 #10

類神經網路 / 深度學習
Neural Networks

疊加多層神經元網絡，自動提取複雜高階特徵。

台灣產業實務案例：
🏥 長庚醫院眼底病變影像判讀
使用情境：非結構化影像/文本且擁有足夠算力 (GPU)。
